## Medit

A creative WYSIWYG rich text editor for mobile device by javascript.

一个创新型的移动端所见即所得富文本编辑器。

Demo Adress : [Medit Demo](https://echosoar.github.io/medit/demo/demo.html)

***

### Current support 目前支持:

##### Basic function 基础功能:

![add left](https://echosoar.github.io/medit/src/image/add.png) Add To Left 向当前块左部添加内容

![delete](https://echosoar.github.io/medit/src/image/close.png) Delete 删除当前块

![ok](https://echosoar.github.io/medit/src/image/ok.png) Ok 完成当前块编辑

![setting](https://echosoar.github.io/medit/src/image/setting.png) Setting 设置当前块样式

![mode](https://echosoar.github.io/medit/src/image/mode.png) Mode 选择当前块类型

![add right](https://echosoar.github.io/medit/src/image/add.png) Add To Right 向当前块右部添加内容

![Add To Next Line](https://echosoar.github.io/medit/src/image/br.png) Add To Next Line 添加至下一行
